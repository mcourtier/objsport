## Concept

**Le Gym** est la zone de **coaching en petit groupe** (1 à 3 personnes). Elle couvre la préparation physique, la rééducation active et le retour à la performance.

Ce n’est pas une salle ouverte au public : Objectif Sport intervient **sur site ou à domicile**, avec l’équipement et l’encadrement adaptés.

---

## Exemples d’espaces équivalents

- Salle de musculation
- Box crossfit
- Espace de renforcement fonctionnel

---

## Activités

### Préhab (dès 45 min)

| Activité | Description |
| -------- | ----------- |
| Routine activation | Session pour partir du bon pied |
| Rituel prévention | Prévoir c’est gouverner |
| Circuit recovery | Récup active — décrassage |

### Réhab (dès 45 min)

Sessions actives spécifiques : genou, cheville, rachis, épaule.

### Réathlé (dès 45 min)

| Activité | Description |
| -------- | ----------- |
| Return to train | Encadrement reprise terrain |
| Return to perf | Encadrement reprise compétition |
| HIIT | Sessions actives spécifiques |

---

## Modalités

- **Format :** coaching 1–3 personnes, sessions de 45 min minimum
- **Fil éditorial :** coaching, consulting (programmation, prévention)
