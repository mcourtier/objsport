## Concept

**Le Studio** est la zone de **cours en small-groupe** et d’**ateliers pédagogiques**. Elle vise le renforcement, la souplesse, la maîtrise du mouvement et la transmission de bonnes pratiques.

Format collectif mais encadré, adapté au niveau des participants.

---

## Exemples d’espaces équivalents

- Studio de danse
- Salle de pilates
- Salle de gym douce / yoga
- Espace cours collectifs (HIIT, fondamentaux du mouvement)

---

## Activités

### Gymnastique (45 min)

| Activité | Description |
| -------- | ----------- |
| Pilates | Force — souplesse — contrôle |
| Yoga | Harmonie — confiance — sérénité |
| Fondamentale | Les clés du mouvement |
| SIIT | Soft Intensity Interval Training |
| HIIT | High Intensity Interval Training |

### Formations (2 h)

| Atelier | Description |
| ------- | ----------- |
| L’échauffement | L’enseignement de la construction |
| Les étirements | Actifs — passifs — dynamiques |
| La prévention | Prévenir c’est guérir |
| Le repos & la récup | Et si c’était le secret de la perf ? |
| Commotion et sport | Les premiers gestes qui sauvent |

## Modalités

- **Format :** cours small-groupe (45 min) ; ateliers formation (2 h)
- **Fil éditorial :** coaching (animation), consulting (pédagogie, prévention)
