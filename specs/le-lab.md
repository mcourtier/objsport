## Concept

**Le Lab** est la zone de **soins et de testing en séance individuelle**. Elle combine récupération, thérapie manuelle ou instrumentale et bilans de performance mesurables.

Objectif : diagnostiquer, soigner et quantifier la progression du sportif.

---

## Exemples d’espaces équivalents

- Salle de testing (force, vitesse, fonctionnel, cognitif, corporel)
- Cabine ou espace de soins (massage, mobilité, thérapie)
- Zone d’électrostimulation / récupération

---

## Activités

### Soins (dès 45 min)

| Activité | Description |
| -------- | ----------- |
| Massage | Recovery — relax — mobilité |
| Mobilité | Mobilisation — stretching |
| Thérapie | Manuelle — instrumentale — physio |
| Esthétique | Endermologie — nutrition |

### Testing (45 min)

| Activité | Description |
| -------- | ----------- |
| Retour au sport | Return to train — play — perf |
| Force & vitesse | Isocinétisme — puissance — endurance |
| Fonctionnel | FMS — KFS — K.Cart — Ankle.go — rachis |
| Spécifique | Marche — course — sauts — geste |
| Cognitif | Équilibre — réflexes — préférences |
| Corporel | Impédance — anthropo — biométrie |

---

## Modalités

- **Format :** séances individuelles, 45 min minimum
- **Fil éditorial :** soins, consulting (interprétation des bilans, recommandations)
